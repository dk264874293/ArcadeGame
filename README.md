

# 开始我们的小游戏

直接在浏览器中运行index.html就可以开始游戏,会有不同的难度选择等着你。每个难度的怪物形状、速度、数量都不相同哟~欢迎挑战困难模式。

## 配置游戏难易度

在**engine.js**第**171**行有**option**变量他存储了不同难度下的配置项,您可以通过修改里面的参数来调整不同的难度。每个参数上面都有注释说明哟。
